svg-to-swift

Convert SVG path data to swift 3 syntax

In the age of disparate devices and device resolutions, vector graphics have become all too important. If you're an iOS developer, why waste your time managing a million image assets to account for different pixel densities and states? You shouldn’t!

svg-to-swift is a small little tool written in javascript to help you convert your vector SVG images into UIBezierPath shapes. Simple paste the string inside the d attribute and vóila, your UIBezierPath code is generated for you.
